# Gakais Flight Table
<p><img src="http://media-elerium.cursecdn.com/attachments/23/613/banner.png" alt=""></p>
<p>A table, that consumes different kinds of fuel to grant creative-like flying capabilities in a specified range.</p>
<p>Parameters like fuel consumption and ranges can be adjusted in the config file "FlightTable.cfg".</p>
<p><strong>Fuels:</strong></p>
<ul>
<li>Feather (default: 12k RF)</li>
<li>Redstone Feather (default: 48k RF)</li>
<li>Redstone Flux</li>
<li>Creative Feather (infinite fuel, e.g. for adventure maps)</li>
</ul>
<p><strong>Upgrade:</strong></p>
<ul>
<li>You can upgrade the range by inserting up to 64 glowstones into the upgrade slot:<br><img src="http://media-elerium.cursecdn.com/attachments/23/612/flighttable-gui.JPG" alt="" width="331" height="150"> <img src="http://media-elerium.cursecdn.com/attachments/23/614/flighttable-gui-with-upgrade-and-feather.JPG" alt="" width="331" height="150"></li>
</ul>
